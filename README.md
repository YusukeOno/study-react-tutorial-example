## Volta install

```zsh
# install Volta
curl https://get.volta.sh | bash

# install Node
volta install node

# start using Node
node
```

## react-typescript-weather-app

```zsh
npm create vite@latest

Project name:
-> react-typescript-weather-app

Select a framework:
-> React

Select a variant:
-> TypeScript

cd react-typescript-weather-app
npm install
npm run dev
```
